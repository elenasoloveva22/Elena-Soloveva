QA-инженер с практическим опытом тестирования, полученным в ходе выполнения проектов. Владею полным циклом ручного тестирования: провожу анализ требований, составляю тестовую документацию (чек-листы, тест-кейсы, отчёты), выполняю функциональное, регрессионное, UI/UX-тестирование веб и мобильных приложений. Работаю с API через Postman, анализирую сетевой трафик в Charles, использую DevTools для отладки интерфейса. Дополнительно реализую автотесты на Python (Selenium WebDriver, Pytest, Page Object, Requests), что позволяет оптимизировать проверки регресса. Готова применять и развивать свои навыки, обеспечивая качество продукта за счет сбалансированного сочетания ручных и автоматизированных подходов. В тестировании я ценю процесс превращения сырого функционала в отполированный продукт. Каждый найденный баг — это маленькая победа, каждый успешный тест-ран — подтверждение, что мы движемся в правильном направлении.
